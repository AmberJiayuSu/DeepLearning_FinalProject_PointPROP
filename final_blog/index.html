Previous Work

Various approaches have been developed for solar radiation estimation. Classical methods like ray-tracing algorithms and radiosity models provide physically accurate results but are computationally expensive. Tools like Ladybug and Honeybee have made these simulations more accessible but still require simplified geometric inputs. Recent machine learning approaches have explored using convolutional neural networks for solar potential estimation, but most work with 2D raster data or require preprocessed mesh geometries. Few methods can directly process raw point cloud data while accounting for temporal variations in solar position and weather conditions throughout the year.

Estimating solar radiation on building surfaces at fine temporal scales (hourly) is important for urban energy and renewable potentially analysis. Traditionally, this is done with physics-based simulation (ray-tracing, GIS tools like Ladybug/Honeybee), which can be computationally intensive for large city models and often require manual simplification of the input 3D geometries to be compatible with the simulation tools. Recent advances in machine learning have shown promise in approximating complex physical processes with data-driven surrogate models. In this study, we propose a deep learning approach to estimate hourly solar radiation on building surfaces using 3D city models and meteorological data as inputs.

This is challenging because raw LiDAR/mesh data are uncleaned – they include millions of points with buildings, trees, noise – and the problem involves geometric occlusion (shadows) varying with time.
